# Examples

HwpForge 사용 예제 모음

## Structure

- `01_basic_units.rs` — HwpUnit, Color 기본 사용법
- `02_document_creation.rs` — Document 생성
- `03_template_usage.rs` — Style Template 적용
- `04_hwpx_roundtrip.rs` — HWPX 읽기/쓰기
- `05_markdown_conversion.rs` — Markdown ↔ HWPX
- `06_hwp5_import.rs` — HWP5 → HWPX 변환

## Running

```bash
cargo run --example 01_basic_units
```
